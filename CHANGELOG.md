# Changelog

All notable changes to the **Open in cmux** extension are documented here.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/), and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2026-05-02

### Added
- Initial release.
- `cmux.openWorkspace` command, available from the Explorer context menu on folders.
- Folder paths are dispatched to the cmux app via the `New cmux Workspace Here` macOS Service.
